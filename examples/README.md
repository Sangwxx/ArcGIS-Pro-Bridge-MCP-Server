# Examples Directory

Files in this directory are intended for first-time users of the project.

If you have the project running but are unsure how to connect a client, ask questions, or test, start here.

## File Descriptions

- `cursor-mcp-config.json` - MCP configuration example for Cursor / Trae style
- `claude-desktop-mcp-config.json` - MCP configuration example for Claude Desktop
- `arcpy-buffer-example.py` - Minimal ArcPy Buffer example
  - Useful for verifying that `execute_arcpy_code` is working
- `prompt-examples.md` - Example prompts for AI MCP clients
  - Suitable for GIS users unfamiliar with this workflow to copy and use directly
